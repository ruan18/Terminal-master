# Terminal Master

🎯 Suíte modular de scripts avançados para automação em terminal Linux.  
Desenvolvido para profissionais de DevOps, pentest, automação e manutenção de sistemas.

## 🚀 Recursos principais

- 📊 Healthcheck com log criptografado
- 🔐 Integração com Telegram para alertas
- 📦 Instalador automático com dependências
- 🧠 Scripts em Bash e Python modulares
- 📁 Organização avançada de logs
- 🛠️ Ferramentas CLI prontas para produção
- 📜 Documentação técnica profissional
